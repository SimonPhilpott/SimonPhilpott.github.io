Work examples

Filterable, sortable tables

    • This projects goal was to extend the user interaction for data tables on a legacy content management system, with very minimal administration required. Requirements were to allow staff to quickly find (in the case of business use) documents contained within a table and apply filters to the table based on a series of meta data fields. Specification stated the use of vanilla Javascript with no reliance on libraries.

    Extension to this functionality later included sortable columns, specific special character handling, URL pre-filtering on page loading (depending on the custom URL) and the requirement to port this functionality over to a future platform

    Responsive tooltips

    • As part of an upgrade to a business service delivery model an extension to the standard HTML tooltips were required. Tooltips needed to remain responsive to client viewing widths and to contain more content than just standard text. Specification stated the use of vanilla Javascript with no reliance on libraries.

    Later requirements also included a timed fade out and support for different content page placing and on the fly client window resizing.

    Dynamic page bookmark menu

    • This piece of development was produced to support a content management project. Senior management strongly wished to address the business perception of a specific intranet area from loose content writing style, into a focused guidebook that became a more solid part of work ethic.

    The scope of this project was to introduce new tools to a legacy intranet to support users in navigating at times, very large pages of content. I developed a dynamically created page navigation menu that required minimal administration to implement which supported pre-existing content formatting. Specification stated the use of vanilla Javascript with no reliance on libraries.

    The menu would be traditionally fixed position, rather than absolute.
